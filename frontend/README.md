# NoteFlow - Frontend

This is the React-based frontend for the Notes Management System.

## Quick Links
- **Live App:** [https://noteflow-app-seven.vercel.app/](https://noteflow-app-seven.vercel.app/)
- **Backend Repo:** [GitHub Repository](https://github.com/maniksharma22/notes-management-system)

## Features
- Real-time Note Creation
- Responsive UI
- Seamless API Integration

## Run Locally
1. `npm install`
2. `npm start`
