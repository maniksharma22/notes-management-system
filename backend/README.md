# Notes Management Backend

This directory contains the Spring Boot application logic for the Notes Management System.

## Project Structure
- **notes-management/**: The core Maven project containing source code and configurations.
- **src/**: Java source files and application resources.

## Setup
To run this service locally, ensure you have Java 17 and Maven installed, then run:
`mvn spring-boot:run` from within the `notes-management` directory.
